# the output of various print statements
---

```py
>>> print("Hello, world!")
Hello, world!
```


```py
>>> print("Hello", "world!")
Hello world!
```


```py
>>> print(3)
3
```


```py
>>> print(3.0)
3.0
```


```py
>>> print(2 + 3)
5
```


```py
>>> print(2.0 + 3.0)
5.0
```


```py
>>> print("2" + "3")
23
```


```py
>>> print("2 + 3 =", 2 + 3)
2 + 3 = 5
```


```py
>>> print(2  3)
6
```


```py
>>> print(2  3)
8
```


```py
>>> print(7 / 3)
2.3333333333333335
```


```py
>>> print(7 // 3)
2
```
